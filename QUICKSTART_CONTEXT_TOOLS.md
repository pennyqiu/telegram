# 🚀 Token-Friendly 上下文工具快速开始

> **一行命令激活，任何环境都能用！**

## ⚡ 超快安装（30秒）

### **macOS / Linux**
```bash
git clone <your-repo> && cd telegram
./setup_context_tools.sh
source .context_aliases_local
```

### **Windows**  
```cmd
git clone <your-repo>
cd telegram
setup_context_tools.bat
context_aliases.bat
```

## 🎯 立即使用

```bash
# 切换到订阅系统开发
ctx-sub
# ✅ 已切换到：📱 Telegram 订阅系统

# 切换到投资工具开发
ctx-inv  
# ✅ 已切换到：💰 投资工具集

# 查看当前状态
ctx-status
# 🎯 当前模式：💰 投资工具集

# 回到全部模块
ctx-all
```

## 🔄 在任何新环境使用

1. **克隆仓库**: `git clone <your-repo>`
2. **运行安装**: `./setup_context_tools.sh` (或 `.bat`)  
3. **激活别名**: `source .context_aliases_local`
4. **开始使用**: `ctx-help`

## 💡 为什么这样设计？

- **🌍 跨平台**: Windows/Linux/macOS 一套代码
- **📦 便携性**: 跟随 Git 仓库，环境无关
- **⚡ 零配置**: 一键安装，立即可用
- **🔒 安全性**: 不修改系统文件，仅项目内有效
