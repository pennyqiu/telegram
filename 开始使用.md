## 💡 实战建议

### 建议1：从少量开始
```
❌ 一开始就想收集100个产品
✅ 先深入研究2-3个最感兴趣的产品
✅ 逐步扩展到5-10个产品
✅ 保持定期更新
```

### 建议2：按需填写
```
储蓄险重点：
  ✓ 收益分析（保证/非保证收益）
  ✓ 现金价值（第5/10/20年）
  ✓ 分红实现率历史
  ✓ 回本年期

重疾险重点：
  ✓ 保障范围和病种
  ✓ 多次赔付机制
  ✓ 理赔网络
  ✓ 保额递增

医疗险重点：
  ✓ 保额上限
  ✓ 免赔额
  ✓ 直付服务
  ✓ 全球网络
```

### 建议3：对比维度
```
同类产品横向对比：
  □ 费用：年缴保费、总保费
  □ 收益：预期收益率、回本期
  □ 风险：公司评级、分红实现率
  □ 流动性：退保损失、保单贷款
  □ 适配度：是否匹配家庭需求
```

## 💬 常见问题

**Q: 工具需要安装什么软件？**
A: 只需要Python 3（Mac自带），无需额外安装。高级功能需要pandas，但不是必需的。

**Q: 我应该研究多少个产品？**
A: 建议每个类别2-3个产品即可，总共5-10个产品。

**Q: 填写一个产品需要多久？**
A: 快速版30分钟，完整版1-2小时。建议分批次完成。

**Q: 如何获取产品说明书？**
A: 
1. 官网下载（最快）
2. 致电客服索取
3. 通过经纪人获取

**Q: 对比后怎么决策？**
A: 
1. 根据对比结果筛选2-3个候选产品
2. 预约持牌经纪人深入咨询
3. 核对最新信息
4. 赴港签约
