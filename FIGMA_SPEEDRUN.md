# Figma 速成路径：3 小时出 9 张小红书图

> 配套文档：`小红书01_港险合法性.md`（及后续小红书系列）
>
> 目标：从 0 基础开始，3 小时内完成 1 篇文章的 9 张配图设计，并搭建好可长期复用的模板库。

---

## 总览：3 小时如何分配

```
准备工作        10 分钟  → 注册 + 装字体
Hour 1 基础     60 分钟  → 看 1 个入门视频 + 跟着练
Hour 2 搭组件库  60 分钟  → 建立你的"设计资产"
Hour 3 出 9 张图 50 分钟  → 灌入文案 + 导出
```

---

## 准备工作（10 分钟）

### 1. 注册 Figma（3 分钟）

- 打开 https://www.figma.com
- 用 Gmail / 邮箱注册免费个人版
- **不需要下载**，浏览器版本完全够用（推荐 Chrome / Edge）
- 可选：装 Mac 桌面客户端（更稳定）

### 2. 装中文字体（5 分钟）

你需要这两款字体（都免费）：

```
1. 思源黑体 Source Han Sans
   下载：https://github.com/adobe-fonts/source-han-sans/releases
   选 OTF/SuperOTC/SourceHanSansSC.otf（简体中文）

2. 阿里巴巴普惠体 3.0
   下载：https://fonts.alibabagroup.com/
   全套下载，双击安装
```

装完 **重启浏览器**，Figma 才能识别。

### 3. 装一个必备插件（2 分钟）

- Figma 顶部菜单 → Plugins → Browse all plugins
- 搜索安装：**"Iconify"**（免费图标库，覆盖你需要的"持牌徽章、警告标识、对勾叉号"）

## Hour 1：基础操作（60 分钟）

### 看视频（30 分钟）

**B 站搜索关键词**（按顺序优先看）：

```
"Figma 30分钟入门"      ← 首选这个
"Figma 零基础教程"
"Figma 小红书封面"      ← 看 1 个对应实战
```

**怎么挑视频（重要）**：

- ✅ 时长 **20-40 分钟**（太长低效，太短不全）
- ✅ 标题含"零基础/入门/新手"
- ✅ 播放量 > 5 万（质量过滤）
- ✅ UP 主有连续的 Figma 系列（说明专业）
- ❌ 跳过：标题含"高级/进阶/UI 设计师必看"

**推荐 UP 主关键词**（搜索时加这些名字优先）：

- "Mockplus""即时设计""设计师阿威""Genji 是真想教会你"

### 边看边操作要掌握的 7 个动作（30 分钟）

| 动作 | 快捷键 | 用途 |
|---|---|---|
| 新建画框 Frame | `F` | 这是你的"画布"，1 个 Frame = 1 张图 |
| 文字 Text | `T` | 写标题、正文 |
| 矩形 Rectangle | `R` | 做卡片底、色块 |
| 移动 Move | `V` | 选中和移动 |
| 缩放 Scale | `K` | 等比缩放 |
| 对齐 Align | 选中后看右侧面板 | 居中、分布 |
| 复制 Duplicate | `Cmd+D` | 复制并保留间距，关键效率技巧 |

**实战练习**：跟着视频做一张任意小红书封面，**不追求好看，追求操作熟练**。

## Hour 2：搭建你的组件库（60 分钟）

这是 Figma 真正的杀手锏。做完这一步，未来你每篇文章节省 80% 时间。

### Step 1：新建项目（2 分钟）

- 左上角 → New design file → 重命名为 **"小红书港险系列_模板库"**

### Step 2：设置颜色样式（8 分钟）

1. 用矩形工具 `R` 画一个色块
2. 右侧面板 Fill → 输入色号 `#0A2540`
3. 点 Fill 旁的四个点图标 → "+ Create style" → 命名 **"主色/深海军蓝"**
4. 重复上述步骤建立这 5 个颜色：

```
主色/深海军蓝   #0A2540
辅色/纯白       #FFFFFF
辅色/米白衬底   #F5F1E8
警示/红         #E63946
强调/暗金       #C9A961
```

> 完成后，你右侧 Local styles 面板会有 5 个颜色圆点。之后任何文字 / 形状都能一键调用。

### Step 3：设置文字样式（10 分钟）

1. 用 `T` 写一个文字 → 选中 → 右侧 Text 面板：
   - 字体：思源黑体 Heavy
   - 字号：120
2. 点 Text 旁的四个点 → "+ Create style" → 命名 **"H1/主标题"**
3. 建立这 4 个文字样式：

```
H1/主标题      思源黑体 Heavy   120pt   行距 1.2
H2/副标题      思源黑体 Bold     56pt    行距 1.4
Body/正文      思源黑体 Medium   42pt    行距 1.6
Caption/角标   思源黑体 Regular  28pt    行距 1.4
```

### Step 4：建 5 个 Master 版式（35 分钟，最关键）

每个 Frame 尺寸都是 **1242 × 1660 px**（按 `F` → 右侧面板手动输入宽高）。

#### 版式 1：封面（图 1 用）

```
- 整个 Frame 填充深海军蓝 #0A2540
- 上半部：H1 大字"港险，到底合不合法？" (白色)
- 中部：H2 副标题"CPA 视角的 5 条边界" (米白色)
- 右下角：角标 "✓ HKIA 持牌经纪" + Iconify 插件搜 "shield-check"
```

#### 版式 2：钩子页（图 2 用）

```
- 深蓝底
- 上方：超大字"90%" (暗金色 #C9A961，字号 240)
- 中部：H2"的人都搞错了"
- 底部：小字 "往下翻 →" + 一个向下箭头
```

#### 版式 3：结论金句页（图 3 用）

```
- 上部 1/4：米白色色块带"✅ 一句话答案"
- 中间 1/2：白底 + 大字 H1 金句
- 下部 1/4：深蓝底 + 白字补充说明
```

#### 版式 4：边界卡片页（图 4-7 共用，这个最重要）

```
- 顶部 80px 处：小字 "5 条边界 · 第 X 条" (角标样式)
- 紧接 H1：⚠️ 边界 X：xxx
- 中间：4 行对比清单（× 红色 / ✓ 绿色 + 文字）
- 底部：浅色卡片 "CPA 提示：xxx"
- 整体留白 80px 边距
```

> **关键操作**：做完这个版式，**右键 → "Create component"**（快捷键 `Cmd+Option+K`），把它变成可复用组件。后续 4 张边界卡只需复制 + 改文字。

#### 版式 5：CTA 引流页（图 8-9 共用）

```
- 上部：H1"6 条避坑要点（建议收藏）"
- 中间：6 行编号列表（数字用暗金色突出）
- 底部：深蓝色块 + 白字 CTA"评论区扣【30问】"
```

### Step 5：右下角全局角标（5 分钟）

```
画一个小元素：
   "X 会计师 · HKIA 持牌经纪 · CPA"
   字号 24pt，米白色，半透明 70%
   放在每个 Frame 右下角 60px 边距处
```

**右键 → Create component**，命名 **"全局角标/V1"**。**这是你的视觉锚点**，每篇文章每张图都用同一个。

## 视觉规范速查（统一标准）

```
画布规格：1242 × 1660 px（小红书 3:4 推荐，清晰度最高）

主色：#0A2540（深海军蓝，类似高盛/麦肯锡报告）
辅色：#FFFFFF（纯白）+ #F5F1E8（米白衬底）
警示色：#E63946（仅用于"违法/违规"标记，慎用）
强调色：#C9A961（暗金，仅用于"持牌印章 / CPA"角标）

字体（中文）：
- 标题：思源黑体 Heavy / 阿里巴巴普惠体 Heavy
- 正文：思源黑体 Medium / OPPO Sans Regular
- 数字/英文：Inter Bold / Roboto Mono（增强专业感）

字号层级（基于 1242px 宽度）：
- 主标题：96-120pt
- 副标题：48-56pt
- 正文：36-42pt
- 角标/页码：24-28pt

栅格：左右各留 80px 边距，元素间距 48px 倍数
```

## 常见卡点 & 解决方案

| 卡点 | 解决 |
|---|---|
| 中文字体在 Figma 里不显示 | 装完字体 **关闭并重启浏览器**；如果用网页版且字体没识别，下载 Figma 桌面客户端 |
| Frame 比例不对 | 永远先按 `F` 然后 **右侧手动输入 1242 × 1660**，不要拖拽生成 |
| 复制后样式变了 | 检查文字是否用了 Text style，没用的话每次复制都要重设 |
| 图标找不到 | Plugins → Iconify → 搜英文关键词（shield-check / warning / cross-mark） |
| 导出后图模糊 | Export 选 **2x**，不是 1x |
| 时间超过 3 小时 | 跳过 Hour 2 的"Create component"步骤，先用复制粘贴；下次再补 |
