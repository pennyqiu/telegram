# 投资工具集合

> 根目录下散布的投资分析与学习工具

## 工具列表

### 📊 分析工具
- **`stock-valuation-calculator.html`** - 股票估值计算器（DCF、PE、PEG等模型）
- **`investment-tracker.html`** - 投资组合跟踪表
- **`damodaran_analysis.html`** - 达摩达兰估值分析工具

### 📚 学习资源  
- **`us-stock-learning.html`** - 美股投资学习指南
- **`static/stock-learning.html`** - 股票学习静态版本
- **`value_investing_analysis.html`** - 价值投资分析框架
- **`farnam_analysis.html`** - Farnam Street 投资思维

### 📑 数据文件
- **`scraped_articles.json`** - 抓取的投资文章数据
- **`acquired-microsoft-summary-cn.md`** - 微软收购案例分析

## 使用方法
```bash
# HTML工具 - 直接浏览器打开
open stock-valuation-calculator.html
open investment-tracker.html

# Python脚本工具见 investment_tracker/ 目录
```

## 特点
- **纯前端**：HTML工具无需后端，本地即可使用
- **独立性**：各工具互不依赖，可单独使用  
- **实用性**：涵盖估值、追踪、学习全流程

**💡 提示**：这些是独立的投资辅助工具，与 Telegram 系统无关。
