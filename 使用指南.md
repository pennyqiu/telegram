# 保险产品研究对比工具 - 使用指南

## 🎯 工具概述

这套工具帮助您系统化地研究和对比不同保险产品，基于《港险投保30问》的知识框架设计。

## 🚀 快速开始（3步走）

### 第1步：查看模板结构


**模板包含的关键信息：**
- 基本信息（产品名称、公司、类型、币种）
- 费用结构（年缴、总费用、缴费年期）
- 保障内容（保额、保障范围、特色功能）
- 收益分析（保证/非保证收益、现金价值、回本年期）
- 流动性（退保损失、保单贷款、减额缴清）
- 理赔服务（理赔网络、时效、免赔额）
- 税务与传承（CRS、受益人、跨境影响）
- 公司评级（历史、评级机构、分红实现率）
- 适用场景（推荐人群、配置优先级）
- 优缺点分析（主要优势/劣势、竞品对比）
- 个人评估（综合评分、是否考虑购买、决策要点）

### 第3步：填写产品信息并对比

#### 3.1 创建产品文件

```bash
# 复制模板
cp insurance_template.json AXA_盛利II_20260520.json
cp insurance_template.json 保诚_信守明天_20260520.json
cp insurance_template.json 平安_守护百分百_20260520.json
```

#### 3.2 使用任何文本编辑器填写

**重点字段（必填）：**
- 产品名称、公司、类型
- 年缴保费、总缴费、缴费年期
- 基本保额
- 预期收益率（储蓄类）
- 退保损失率
- 个人评分和评估

**可选字段：**
- 根据产品类型选择性填写
- 储蓄险重点填"收益分析"
- 重疾险重点填"理赔服务"
- 不相关的可以删除或标注"不适用"

#### 3.3 对比产品

```bash
python3 simple_compare.py
```

**可用功能：**
1. **产品摘要** - 快速查看所有产品概况
2. **储蓄类产品对比** - 横向对比收益、费用、回本期
3. **重疾险产品对比** - 对比保额、保费、理赔时效
4. **查看产品详情** - 查看单个产品完整信息
5. **家庭保险配置分析** - 输入家庭情况，获取配置建议
6. **导出对比报告** - 生成文本格式的完整报告

## 💡 实用建议

### 阶段一：理论学习（1-2天）

- [x] 您已完成《港险投保30问》
- [ ] 理解5大产品类型的区别
- [ ] 掌握保证收益 vs 非保证收益
- [ ] 了解分红实现率的重要性
- [ ] 理解CRS和税务影响

### 阶段二：产品调研（1-2周）

**建议策略：**
1. 先确定需求优先级
   - 医疗保险（高优先级）
   - 重疾险（高优先级）
   - 储蓄分红险（中优先级）
   - 年金险（低优先级）

2. 每个类别选2-3个产品深入研究
   - 不要贪多，先从最感兴趣的开始
   - 完整填写模板，不要跳过分析部分

3. 重点关注的信息
   - **储蓄险**：分红实现率历史、回本年期、第20年现金价值
   - **重疾险**：保障范围、多次赔付、保额递增机制
   - **医疗险**：理赔网络、直付服务、免赔额

### 阶段三：横向对比（3-5天）

1. 使用对比工具生成表格
2. 重点对比：
   - 同类产品的性价比
   - 保费与保障的平衡
   - 公司的历史分红实现率
   - 流动性和退保损失

3. 标注每个产品的优缺点
4. 填写"个人评估"部分

### 阶段四：家庭规划（1-2天）

1. 评估家庭财务状况
   - 年收入
   - 家庭结构
   - 现有保障
   - 风险承受能力

2. 使用工具的"家庭配置分析"功能
3. 制定保险配置方案
4. 计算总保费预算（建议：年收入的10-15%）

## 💬 常见问题

### Q1: 我应该研究多少个产品？
**A:** 建议每个类别2-3个即可。例如：
- 储蓄险：AXA盛利II + 保诚信守明天 + 友邦充裕未来
- 重疾险：AXA挚爱保 + 保诚守护健康 + 平安守护百分百

### Q2: 填写模板需要多长时间？
**A:** 
- 快速版：30分钟（填写关键字段）
- 完整版：1-2小时（包含深度分析）
- 建议：先快速填写，后续逐步完善

### Q3: 没有某些数据怎么办？
**A:** 
- 标注"暂无数据"或"待查询"
- 联系保险公司客服询问
- 咨询持牌经纪人
- 部分字段可以空着或删除

### Q4: 如何获取分红实现率数据？
**A:** 
- 香港保险公司每年公布，查看官网
- 向经纪人索取历史数据
- HKIA网站可能有汇总信息

### Q5: 工具能自动计算IRR吗？
**A:** 
- 当前版本需要手工填写
- 可以使用Excel的IRR函数计算
- 或向保险公司索取演示书（含IRR）
