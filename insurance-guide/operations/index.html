⚙️ 工具与流水线

把 articles/ 下的 HTML 一键转换成公众号 + 知乎双格式

1
在 insurance-guide/articles/ 里挑一篇要发布的文章，例如 cpa-02-fulfillment-ratio.html
2
在 项目根目录运行：
python insurance-guide/operations/tools/convert_single.py "insurance-guide/articles/cpa-02-fulfillment-ratio.html" "港险分红实现率拆解"
3
在项目根目录得到两个文件：公众号_港险分红实现率拆解.html + 知乎_港险分红实现率拆解.md，把它们移入 operations/published/wechat 和 operations/published/zhihu
4
公众号：浏览器双击 HTML → Ctrl+A → Ctrl+C → 粘贴到公众号编辑器（样式全保留）
知乎：直接打开 MD 复制内容粘贴到知乎"写文章"（知乎原生支持 Markdown）
