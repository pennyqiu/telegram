# 04 · 从入门到专业的学习路线图

> 分阶段、可执行的成长路径，帮助新人系统性地建立能力体系

---

## 总体节奏概览

```
第 1–3 个月        第 4–6 个月        第 7–12 个月        第 2–3 年
   ↓                   ↓                   ↓                  ↓
[考牌 + 入门]    [产品 + 首批客户]    [系统化 + 成长]   [专业化 + 晋升]
打好地基          开始产粮              稳定出单           建立口碑
```

---

## 第一阶段：考牌与入门（第 1–3 个月）

### 核心目标：取得执照，建立知识框架

#### 1.1 必考执照

**优先级排序：**

```
Paper 1（保险原理）→ 必须首先完成
       ↓
Paper 3（长期保险）→ 人寿/储蓄/医疗产品的销售资格
       ↓
Paper 5（投资相关）→ 若需要销售 ILAS，后续补考
```

**备考建议：**
- 使用 IA 官网提供的考试大纲和样题
- 参加公司提供的内部培训课程（通常免费）
- Paper 1 + Paper 3 建议在入职后 6–8 周内完成
- 每日学习 1.5–2 小时，约 4–6 周可完成备考

#### 1.2 入门知识清单

| 类别 | 学习内容 | 资源 |
|------|----------|------|
| 监管框架 | IA 网站，《保险业条例》概要 | ia.org.hk |
| 公司产品 | 熟读安盛主力产品条款（储蓄险、医疗险各 2–3 款） | 公司内网 |
| 销售流程 | 需求分析表（FNA）填写方法，投保流程 | 公司培训 |
| 合规底线 | 10 大合规禁忌，冷静期规定 | 主管辅导 |
| 竞品对比 | 了解友邦、保诚同类产品的基本特点 | 市场公开资料 |

#### 1.3 第一阶段的里程碑

- [ ] Paper 1 通过
- [ ] Paper 3 通过
- [ ] 熟背公司 3 款主力产品的核心卖点
- [ ] 完成第一份模拟需求分析
- [ ] 建立自己的准客户名单（目标 50 人）

---

## 第二阶段：产品深化与首批客户（第 4–6 个月）

### 核心目标：开单，积累实战经验

#### 2.1 产品学习深化

| 产品类型 | 深度学习要点 |
|----------|------------|
| **储蓄险** | 保证 vs 非保证回报的计算方法；IRR（内部回报率）的实际计算；与银行定存/基金的比较框架 |
| **医疗险（VHIS）** | 标准计划 vs 自订计划；附加保障（Rider）选择逻辑；理赔实例和流程 |
| **危疾险** | 覆盖病种数量；定义宽严（多重赔付 vs 单次赔付）；早期危疾的价值 |
| **人寿险** | 终身 vs 定期的适用场景；保单贷款功能；受益人安排的税务含义 |

#### 2.2 销售技能基础

**建议学习的销售框架：**

```
FACT 收集：Family / Activities / Cash / Time
       ↓
需求分析：保障缺口 + 财务目标
       ↓
方案呈现：2 选 1 方案，而非单一推荐
       ↓
异议处理：价格/时机/需要考虑
       ↓
促成：认同需求 + 文件跟进
```

**TOP 3 最常见客户异议与应对：**

| 异议 | 应对思路 |
|------|----------|
| "我要再想想" | 了解具体顾虑，将"想想"具体化 |
| "回报太低" | 用 IRR 对比，强调保证部分的稳定性 |
| "我已经有保险了" | 做保障检视（Insurance Review），找出保障缺口 |

#### 2.3 首批客户的合理预期

- 前 3 个月客户主要来自**个人社交圈**（家人、朋友、前同事）
- 目标：第 4–6 个月每月出单 **2–4 件**
- 重视每次成交后的**转介绍（Referral）请求**

## 第三阶段：系统化成长（第 7–12 个月）

### 核心目标：建立可持续的客户开拓系统

#### 3.1 专属客群定位

新人最常见的错误是"见人就说保险"，正确做法是**选择 1–2 个核心客群深耕**：

| 客群 | 核心需求 | 接触渠道 |
|------|----------|----------|
| 大湾区访港内地客 | 美元储蓄 + 医疗保障 | 微信、口岸、转介 |
| 香港年轻专业人士 | 危疾 + 医疗 + 退休规划 | LinkedIn、专业协会 |
| 中小企业主 | 团体医疗 + 关键人保险 | 商会、B2B 转介 |
| 新移民家庭 | 子女教育储蓄 + 全家医疗 | 社区活动、同乡会 |
| 高净值人士 | 遗产规划 + 大额储蓄 | 专业转介（会计师、律师）|

#### 3.2 内容营销入门

建立专业形象，让客户主动找上门：

| 平台 | 内容方向 | 频率建议 |
|------|----------|----------|
| 微信朋友圈 | 保险知识科普、市场动态、客户案例 | 每周 3–5 条 |
| 微信公众号 | 深度文章：如何选择医疗险、储蓄险 IRR 解析 | 每月 2–4 篇 |
| 小红书 | 香港保险避坑指南、真实理赔故事 | 每周 1–2 篇 |
| LinkedIn | 专业形象、行业分析（针对专业人士客群）| 每周 1 条 |

#### 3.3 建立转介绍系统

```
每次成交后必做的 3 件事：

1. 在冷静期后发送感谢信 + 提醒保单激活事项
2. 3 个月后主动跟进客户体验
3. 在适当时机（而非立刻）提出："如果你觉得我的服务还不错，
   是否可以介绍 1–2 位有类似需求的朋友认识我？"
```

#### 3.4 进阶学习资源

| 资源 | 说明 |
|------|------|
| **LOMA 130/280** | 国际认可的保险管理认证 |
| **ChFC（特许财务顾问）** | 全面财务规划认证，提升专业深度 |
| **CFP（认证理财规划师）** | 香港认可，开拓高净值市场的敲门砖 |
| **MDRT 申请** | 达标后即可申请，国际认可的销售成就 |
| **IA CPD 课程** | 每年维持牌照所需的持续进修学时 |

## 第四阶段：专业化与晋升（第 2–3 年）

### 核心目标：建立专业口碑，考虑管理或专家路径

#### 4.1 两条职业路径

**路径 A：销售专家路径**
```
普通代理人 → MDRT → COT（百万圆桌顶级会员）→ TOT（顶级顶级会员）
             年佣金约                年佣金约              年佣金约
             HK$100万               HK$300万              HK$600万+
```

**路径 B：管理发展路径**
```
代理人 → 资深代理人 → 主任（Unit Manager）→ 区域经理 → 高级管理层
         建立团队          管理 5–15 人          管理多个团队
```

#### 4.2 高级产品专项学习

| 专项领域 | 适用客群 | 学习资源 |
|----------|----------|----------|
| 遗产规划与家族信托 | 高净值客户 | 香港律师公会培训 |
| 企业保险方案 | 中小企业主 | LOMA 专业课程 |
| 跨境税务规划 | 内地高净值客户 | 四大会计师事务所的公开讲座 |
| 退休与年金规划 | 临近退休客户 | MPF 顾问培训 |

## 每日/每周学习习惯建议

### 每日 30 分钟

```
早上 15 分钟：读一篇保险行业新闻（IA 官网、经济日报保险版）
晚上 15 分钟：复盘当天客户对话，记录问题与改进点
```

### 每周固定任务

| 频率 | 任务 |
|------|------|
| 每周一 | 制定本周客户拜访/联系计划 |
| 每周三 | 参加公司内部培训或产品分享会 |
| 每周五 | 整理本周客户跟进记录，更新 CRM |
| 每两周 | 阅读一份行业报告或竞品分析 |
| 每月 | 完成一个新知识点的深度学习，并整理成可分享的内容 |
