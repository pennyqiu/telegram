# 02 · 香港保险市场现状与未来展望

> 以数据和趋势为基础，帮助从业者建立市场宏观视野

---

## 一、香港保险市场规模概览

### 1.1 市场体量

香港是亚太区最重要的保险中心之一，也是全球人均保险密度最高的地区之一：

| 指标 | 数据（近年） | 备注 |
|------|------------|------|
| 保险业总保费收入 | 约 **6,000 亿港元/年** | 长期保险占主体 |
| 长期保险新造保费 | 约 **4,500–5,000 亿港元/年** | 含内地访客贡献 |
| 人均保险密度 | 全球前 3 名 | 每人每年保费支出极高 |
| 持牌保险公司数量 | 约 **160 家** | 覆盖全球主要保险集团 |
| 注册保险中介人 | 超过 **11 万人** | 代理人 + 经纪人合计 |

### 1.2 内地访客保费贡献

这是理解香港保险市场不可忽视的核心数据：

| 时期 | 内地访客新造保费 | 占香港个人业务比例 |
|------|---------------|------------------|
| 2019 年（疫情前高峰） | 约 **680 亿港元** | 约 26% |
| 2020–2022 年（封关期间） | 近乎归零 | <1% |
| 2023 年（通关恢复） | 约 **590 亿港元** | 迅速反弹 |
| 2024 年（预估） | 超过 **700 亿港元** | 创历史新高趋势 |

> **关键洞察：** 内地客户需求是结构性的、长期的，而非周期性波动，这为具备普通话沟通能力的从业者创造了巨大机遇。

## 二、当前市场现状分析

### 2.1 主要增长驱动力

**① 内地高净值客户的资产配置需求**
- 人民币资产贬值预期驱动美元资产配置
- 香港保险作为合法的跨境财富管理工具
- 遗产规划、子女教育基金需求旺盛

**② 香港本地医疗保险需求上升**
- 公立医院轮候时间长，私家医疗费用高企
- 自愿医保计划（VHIS）政策推动下普及加速
- 危疾发病率上升，保障型产品需求增加

**③ 人口老龄化带来的退休规划需求**
- 强积金（MPF）积累不足以支撑退休生活
- 年金保险、储蓄保险成为补充退休收入的重要工具
- 长者医疗券计划间接刺激保险意识

**④ 利率环境变化**
- 2022–2024 年高息环境使储蓄险吸引力上升
- 定息保险产品（Guaranteed Return）备受关注

### 2.2 当前市场挑战

| 挑战 | 说明 |
|------|------|
| 从业人员竞争激烈 | 超过 11 万名中介，新人开拓客户难度大 |
| 误导销售问题 | 历史遗留问题，监管持续加码，合规成本上升 |
| 客户教育程度提升 | 网络信息透明，客户比较能力强，价格战明显 |
| 内地政策不确定性 | 内地偶发限制跨境保险购买的政策风险 |
| 新冠后客户信心修复 | 部分客户对长期储蓄承诺仍有顾虑 |

### 2.3 竞争格局：主要保险公司

| 公司 | 市场定位 |
|------|----------|
| **友邦（AIA）** | 市场龙头，品牌最强，代理人体系完善 |
| **安盛（AXA）** | 全球品牌，产品线丰富，高净值客户优势 |
| **宏利（Manulife）** | 北美背景，投资产品见长 |
| **保诚（Prudential）** | 英国系，危疾和医疗口碑佳 |
| **富通（FWD）** | 本土新兴，数字化程度高，定价激进 |
| **中国人寿（COLI）** | 内地背景，服务内地客群优势明显 |

## 三、未来展望（2025–2030）

### 3.1 内地访客市场持续扩张

**驱动因素：**
- 大湾区融合加深，跨境人员流动常态化
- 内地中产和高净值人群数量持续增长
- 人民币国际化进程中，美元资产需求刚性存在
- 遗产税立法预期加速财富规划需求

**预测：** 到 2030 年，内地访客保费有望突破 **1,000 亿港元/年**。

### 3.2 数字化转型加速

| 趋势 | 具体表现 |
|------|----------|
| 保险科技（InsurTech） | AI 核保、自动理赔、数字化投保流程 |
| 客户自助服务 | App 管理保单、在线理赔申请普及 |
| 大数据精准营销 | 客户画像分析，精准推荐产品 |
| 电子签约 | 减少纸质流程，部分产品实现远程投保 |

> **对从业者的影响：** 纯靠"关系销售"将逐步失效，具备数字化工具使用能力的从业者将脱颖而出。

### 3.3 监管趋势：合规要求持续提升

- **RBC（风险为本资本制度）**：保险公司将面临更严格的资本要求
- **KYC/AML 强化**：反洗钱审查升级，尤其针对大额保单
- **产品信息披露**：非保证回报的演示规范将更严格
- **中介人持续培训**：CPD 学时要求有望进一步提高

### 3.4 产品创新方向

| 创新领域 | 说明 |
|----------|------|
| ESG 相关保险产品 | 绿色保险、可持续投资连结产品 |
| 大湾区跨境医疗险 | 支持在内地就医报销的医疗保险 |
| 长期护理保险 | 应对老龄化的护理支出保障 |
| 网络安全保险 | 企业数字风险管理需求上升 |
| 参数型保险 | 气候变化相关的创新赔付机制 |

### 3.5 职业前景预测

| 维度 | 判断 |
|------|------|
| 市场整体增速 | 年均 5%–8% 增长，属稳健成长行业 |
| 新人机会窗口 | 内地访客市场为新人创造结构性机会 |
| 收入天花板 | 头部从业者年收入可达数百万港元，分化明显 |
| 职业稳定性 | 纯佣金制度，初期收入不稳定是最大风险 |
| 长期价值 | 续期佣金积累后，职业被动收入价值极高 |

## 四、从业者的市场机遇矩阵

```
                    高潜力
                      ↑
      大湾区跨境客户  |  高净值遗产规划
                      |
  ←————————————————————————————————→
  低竞争                          高竞争
                      |
      本地年轻族群退休规划  | 传统人脉销售
                      |
                      ↓
                    低潜力
```

**建议新人优先聚焦：大湾区跨境客户 + 专业人士群体（律师、医生、IT、会计等）**

---

## 小结

| 核心判断 |
|---------|
| 香港保险市场长期向好，内地访客是结构性机遇 |
| 数字化能力将成为新一代从业者的核心竞争力 |
| 合规意识是长期职业发展的基础与护城河 |
| 产品向保障型 + 储蓄型双轮驱动，专业化程度决定收入上限 |
