海归 PR 的『双重税务陷阱』：3 个真实场景预警

『拿了美国 / 加拿大 PR，但人长居中国，
税务问题是不是不用操心？』

恰恰相反。
PR 身份意味着『全球征税义务』。
不知情 = 双重申报陷阱。

—— 全球征税原则 ——

美国 / 加拿大 / 澳大利亚 / 新加坡：
PR 持有人对全球收入有申报义务

美国特殊：
公民 + PR 都是全球征税
即使长居海外也要申报

—— 陷阱 1 · 全球收入未申报 ——

场景：
某海归 · 美国 PR · 长居深圳 5 年
中国工资 + RSU 在中国已纳税
但未向 IRS 申报美国全球收入

风险：
美国可追溯 6 年
+ 罚款（最低 ＄10000 / 漏报）
+ 滞纳金

正确做法：
✓ 每年 4 月 15 前申报美国 1040
✓ 利用『海外收入豁免』Form 2555
   （≤ 12.6w 美元免税豁免，2024 数据）
✓ 利用『海外税收抵免』Form 1116
   （已在中国纳的税抵扣）
→ 通常实际不重复缴税
→ 但必须申报，不能不报

—— 陷阱 2 · 海外金融账户未披露 ——

美国 FBAR（FinCEN 114）：
任何时候账户余额 > 1w 美元
必须申报

美国 FATCA（IRS Form 8938）：
单身年末 > 5w 美元 / 任何时点 7.5w 美元
已婚联合申报年末 > 10w / 任何时点 15w
必须申报

风险：
未申报 = 严厉罚款
最高可达账户余额 50%（重大故意）
+ 刑事责任（极端）

正确做法：
✓ 每年申报 FBAR + FATCA
✓ 如实披露所有海外账户（含港险现金价值）

—— 陷阱 3 · 资本利得双重征税 ——

场景：
卖出中国 A 股 / 房产 / 股权
中国可能不征（A 股）或低征（房产）
但美国按全球征税
要补缴美国税

避免方法：
✓ 中美税收协定『打破规则』
✓ 申请『海外税收抵免』
✓ 部分情况可适用『豁免条款』

但前提：
要主动申请 + 准确填表
否则默认双重征税

—— PR vs 公民差异 ——

PR 持有人：
放弃 PR 身份后 → 停止全球征税

美国公民：
放弃国籍要做『出境税』(Expatriation Tax)
资产视同卖出
一次性结算资本利得
（高净值人群代价巨大）

—— 中国 PR / 海外身份的『反向陷阱』 ——

如果你：
拿了美国 PR
+ 中国户籍
+ 长居中国 > 183 天

可能同时是：
美国 PR（全球征税）
+ 中国税务居民（按 183 天）

需要：
中美税收协定『打破规则』判断
归属一方为主税务居民

—— CPA 建议 ——

申请 PR 前：
✓ 做完整税务体检
✓ 评估全球征税影响
✓ 决定是否长期持有

持有 PR 期间：
✓ 年度合规申报（FBAR + FATCA）
✓ 利用海外收入豁免 + 抵免
✓ 联合涉外税务师交叉验证

如计划放弃 PR：
✓ 提前 1–2 年规划
✓ 评估『放弃成本』
✓ 美国公民需『出境税』规划

—— 一个真实样本（脱敏） ——

某客户 · 40 岁 · 美国 PR 8 年 · 长居深圳
未申报美国 1040 + FBAR 5 年
RSU 归属价值 200w 美元

风险评估：
美国可追溯 6 年
+ 罚款基础约 5w 美元
+ 滞纳金

补救方案：
申请『流溪程序』(Streamlined Procedures)
（IRS 对未故意未报的减轻方案）
补报过去 3 年 1040 + 6 年 FBAR
+ 海外收入豁免 + 税收抵免

最终：
实际补缴税款约 3w 美元
避免重大罚款

—— 写在最后 ——

PR 不是『身份红利』。
是『全球征税义务』。
