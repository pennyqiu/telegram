如果你符合以下任一情况：
✓ 互联网中层，资产 60%+ 绑自家股票
✓ 计划送孩子赴海外读书
✓ 已申请或计划申请香港优才 / 高才 / 专才
✓ 已有海外身份但缺乏完整资产规划
✓ 跨境家庭 + 美元配置需求

可以申请 1V1 财务诊断。

诊断流程：
Step 1 · 填 KYC 问卷
（家庭结构 / 资产 / 收入 / 跨境身份 / 目标）

Excel 三套：Baseline / Optimized / Stress Test

Step 3 · 出具诊断报告
10–15 页《家庭财富结构优化与风险对冲报告》

Step 4 · 结构化线上会议（Zoom / 腾讯会议）
共享屏幕逐页讲解

承诺：
✓ 全程免费（首次诊断不收费）
✓ 不强推产品
✓ 不约线下喝咖啡
✓ 全程合规（HKIA 持牌 + CPA）
