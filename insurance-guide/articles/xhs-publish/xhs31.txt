港险冷静期与理赔流程：内地人最关心的 5 件事

『港险买了，理赔在内地能不能办？』
『冷静期到底有什么用？』

5 个最常见问题，今天讲清楚。

—— Q1 · 冷静期是什么 ——

定义：
首期保单生效后 21 天
无条件全额退保
（免任何手续费 + 全额返还首期保费）

法律依据：
《保险业条例》强制规定
所有港险标配，无差别

为什么有冷静期？
保护投保人不被销售压力影响
给你时间冷静审阅条款

—— Q2 · 冷静期内可以做什么 ——

✓ 仔细审阅条款全文（不是计划书，是合同条款）
✓ 联合 CPA / 律师 / 第三方做交叉验证
✓ 与代理人 / 经纪人确认细节
✓ 决定退保（无任何成本）
✓ 决定继续（一过即生效）

冷静期常见操作：
1. 拿到电子保单 PDF（通常 1–7 天）
2. 找第三方专业人士审阅
3. 与代理人讨论疑问
4. 决定继续或退保

—— Q3 · 理赔需要什么 ——

重疾险理赔资料清单：
✓ 重疾诊断证明（三甲医院出具）
✓ 病理报告（癌症必须）
✓ 影像学报告（CT / MRI / 超声）
✓ 检查报告 + 既往病历
✓ 理赔申请表（公司提供）
✓ 投保人身份证明 + 银行账户

医疗险理赔资料清单：
✓ 病例首页
✓ 出院小结
✓ 详细费用清单（医院盖章）
✓ 检查报告
✓ 处方 + 药品费明细
✓ 银行账户

—— Q4 · 理赔时长 ——

重疾险：
资料齐全 + 简单案件 → 7–14 工作日
复杂 / 待补充资料 → 30+ 工作日
争议案件 → 60+ 工作日

医疗险：
直付（高端医疗 / VHIS 部分）→ 当场
后报 → 14–30 工作日

—— Q5 · 跨境理赔流程 ——

不需要赴港！

Step 1 · 内地三甲医院诊断 + 治疗
Step 2 · 资料公证
   - 现行：视频公证（不必赴港）
   - 部分公司接受电子公证
Step 3 · 邮寄香港（保险公司 / 代理人协助）
Step 4 · 香港保险公司核保
Step 5 · 拨付到投保人指定账户
   - 香港银行账户 / 内地银行账户都可

实际体验：
全程线上 + 邮寄
不必赴港
理赔款 7–30 天到账

—— 拒赔常见原因 ——

× 投保时未如实告知（最常见！）
   - 已知疾病未披露
   - 体检异常未披露
   - 用药史未披露

× 不在条款定义范围
   - 重疾定义不达标
   - 医疗费用不在保障范围

× 等待期内出险
   - 重疾等待期通常 90 天
   - 等待期内确诊 = 不予赔付

—— 投保关键 · 如实告知 ——

港险采用『最大诚信原则』
任何已知 / 应知的健康问题
都必须主动告知

3 个建议：
✓ 全部体检报告 → 给经纪人看一遍
✓ 既往病史 → 完整列出
✓ 模糊事项 → 主动询问 + 留书面记录

—— 理赔成功率（参考公开数据）——

重疾理赔成功率：≥ 95%（如实告知前提）
医疗理赔成功率：≥ 98%
拒赔的主要原因：未如实告知 / 等待期内 / 定义不符
