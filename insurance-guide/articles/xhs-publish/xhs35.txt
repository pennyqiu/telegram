被 PR 身份反向『卡住』的人：3 个真实场景拆解

『拿了美国 PR 5 年，但人在中国工作生活。
现在反而觉得 PR 是负担。怎么办？』

每月 3–5 条这样的私信。3 个真实场景拆解 ⬇️

—— 场景 1 · 双重税务陷阱 ——

某海归 · 美国 PR 8 年 · 长居深圳

困境：
美国对 PR 全球征税
+ 中国税居也对全球收入有义务
→ 双重申报 + 双重征税风险

后果：✗ 美国未申报 1040 + ✗ 中国未申报海外资产
+ 滚雪球的滞纳金

—— 场景 2 · 维持 PR 的成本 ——

PR 居住要求：
美国：每年 ≥ 6 月
加拿大：5 年内 ≥ 730 天
澳大利亚：5 年内 ≥ 2 年
新西兰：每年 ≥ 184 天

困境：工作中国 vs 维持 PR 必须长居海外
→ 工作 + PR 二选一

—— 场景 3 · 跨境资产调整难 ——

某客户 · 美国 PR · 在美购房 + 持有
卖出 → 美国资本利得税 + 中国可能课税
持有 → 维护成本 + 不能利用 → 进退两难

—— 3 类解套方案 ——

方案 1 · 主动放弃 PR
美国 PR 放弃 → 停止征税
美国公民放弃需『出境税』（资产视同卖出）
适合：长期不回美国 + 资产主要在中国

方案 2 · 灵活居住安排
申请境外居住豁免 / 转长期签证（E-2 / O-1）
适合：事业仍在 PR 国但工作机会在中国
（律师费高，每年合规复杂）

方案 3 · 切换到他国身份
美国 PR → 香港优才 / 高才
✓ 香港不对海外资本利得征税
✓ 保留中国户籍 + 大湾区灵活居住
适合：工作家庭在大湾区 + 不长居美国

—— 真实样本（42 岁 · PR 10 年 · 长居深圳） ——

诊断：PR 维持成本 > 收益

新方案：
1 申请香港高才（已批）
2 落港 + 切换税居
3 美国 PR 主动放弃（1 年规划）
4 美国房产卖出 + 资金调回

3 年后：港税居 · 资产 70% 大湾区 · 跨境合规清晰
