四、正文文案（小红书 ≤ 1000 字）

『我刚收到香港保险公司通知， 我的港险被 CRS 申报回内地。 现在该怎么办？』 每个月有 3–5 条这样的私信。 今天用决策树讲清楚。 —— 第一步 · 理清前提 —— 3 个核心问题： Q1 你是中国税务居民吗？ Q2 你被申报的账户余额是多少？ Q3 收入性质是什么？ —— Q1 · 税务居民身份 —— 中国税务居民标准： ✓ 在中国境内有住所 ✓ 或在境内连续居住 ≥ 183 天 ✓ 户籍 / 长期居住 如果你是中国税务居民 → 全球收入有申报义务 如果你是非中国税务居民（如已切换到香港） → 仅就中国境内来源收入申报 → 海外资产 CRS 报到内地仅是『信息存档』 —— 决策树 1 · 利息 / 分红 —— 被申报内容： 境外银行账户利息 境外股票股息 其他被动收入 中国处理： - 境外个人储蓄利息：通常免征 - 境外股票股息：按 20% 综合所得税 - 已被境外预扣税 → 可抵扣 - 未达起征点 → 通常不需补缴 操作建议： ✓ 主动申报『综合所得 - 利息股息红利所得』 ✓ 提供境外完税凭证 ✓ 申请抵扣 —— 决策树 2 · 资本利得 —— 被申报内容： 境外房产 / 证券卖出收益 中国处理： - 境外资本利得：按 20% 申报 - 已被境外纳税 → 可抵扣 - 未申报 = 漏报风险 操作建议： ✓ 主动申报 ✓ 提供境外完税凭证 ✓ 利用税收协定避免双重 —— 决策树 3 · 港险分红 —— 港险储蓄分红 / 现金价值增值： 中国当前实操： × 没有明确的『储蓄险分红』征税科目 × 多数案例未实际征税 × 但 CRS 已收到信息，理论上可稽查 操作建议： - 资产规模 < 25w 美元（自动门槛下） → 通常不主动申报 → 保留所有保单凭证 - 资产规模 ≥ 25w 美元 → 主动告知税务师 → 评估申报必要性 → 准备应对稽查 —— 3 类处理方式对比 —— 方式 A · 主动补报 ✓ 风险最低 ✓ 可申请滞纳金减免 ✓ 合规姿态最佳 ✗ 立即产生税务负担 方式 B · 等待稽查 ✗ 风险高（一旦稽查 + 罚款） ✗ 滞纳金按日累计 ✗ 信用记录影响 方式 C · 申诉身份 适用： 你认为自己已不是中国税务居民 但被错误判定 → 提供税收协定『打破规则』证据 —— 决策依据 —— 主要看 3 个变量： 1. 你的税务居民身份是否明确？ 2. 你的收入性质是什么？ 3. 已纳税情况？ → 联合涉外税务师做评估 → 不要自己拍脑袋决定 —— 时间窗口 —— 首次被 CRS 申报后： 中国税务通常给 12–24 个月观察期 （部分情况会主动联系核实） 主动申报窗口： ✓ 越早越好 ✓ 滞纳金按日累计 ✓ 主动 = 罚款减免 ✓ 等到被稽查 = 罚款重 + 信用差 —— 一个真实样本（脱敏） —— 某客户 · 中国税务居民 · 港险现金价值 35w 美元 被 CRS 申报回内地（2024 年） 我和涉外税务师评估后建议： 1. 港险分红当前未实际征税路径明确 → 不主动申报 → 但保留所有凭证 2. 同时披露的香港银行账户利息（约 5000 USD） → 主动申报境外利息 → 中国当前对个人境外储蓄利息通常免征 → 但合规姿态保护 3. 持续关注政策变化 → 如未来对储蓄险分红明确征税路径 → 立即配合补报 最终： 未实际产生税务负担 + 合规姿态最佳 + 不会被未来稽查 —— 写在最后 —— CRS 不是『陷阱』。 是『跨境家庭必须用专业方式应对的合规命题』。 下一篇：《中年职业危机的现金流补丁 5 种方案》 —— X 会计师 HKIA 持牌经纪 · 中国 CPA

六、评论区可能问题预判

客户提问应答模板
不申报会被罚吗？看具体情况。被稽查后才补报 = 滞纳金 + 罚款；主动申报 = 通常仅滞纳金。
怎么找涉外税务师？建议找有 CTA + 涉外业务经验的事务所。可在私信中获取 2–3 个推荐。
我已经切换到香港税务居民了？需要提供切换证据（HKID + 居住 + 工作）。建议联合税务师做身份评估。
分散到家人账户能避吗？不能。家人也是税务居民，账户仍会被申报。还可能涉及代持法律风险。
港险还能继续买吗？完全可以。港险合法性 + 法律保护不受 CRS 影响。
