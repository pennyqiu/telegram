帮助您系统化地收集、整理和对比不同保险产品，为家庭保险规划做出更明智的决策。

## 🚀 快速开始

### 1. 安装依赖

```bash
cd /Users/hustqiu/Desktop/telegram
pip3 install -r requirements.txt
```

### 2. 生成产品信息模板

```bash
python3 insurance_comparison.py
# 选择功能 1：生成产品信息模板
```

这会创建 `insurance_template.json`，包含完整的对比框架。

### 3. 填写产品信息

复制模板并重命名，例如：
- `AXA_盛利II_储蓄险.json`
- `保诚_信守明天_储蓄险.json`
- `平安_守护百分百_两全险.json`

按模板填写各项信息（可以逐步完善）。

### 4. 加载和对比产品

```bash
python3 insurance_comparison.py
# 功能 2：加载产品数据
# 功能 3：对比产品
# 功能 5：导出Excel对比报告
```

### 5. 家庭保险配置分析

```bash
python3 insurance_comparison.py
# 功能 4：输入家庭信息，获取配置建议
```

## 📊 对比维度

工具基于《港险投保30问》设计了完整的对比框架：

### 基础维度
- 产品类型、公司背景、费用结构
- 保障范围、理赔网络

### 收益分析（储蓄类）
- 保证收益 vs 非保证收益
- 分红实现率历史数据
- 不同年期现金价值
- 回本年期分析

### 流动性分析
- 退保损失率（第1/3/5年）
- 保单贷款、减额缴清等灵活性
- 保费假期可用性

### 风险维度
- CRS税务影响
- 公司评级和历史表现
- 跨境理赔便利性

### 适配度分析
- 适用人群和场景
- 与其他产品搭配建议
- 个人评分和备注

## 💡 推荐工作流程

### 阶段一：知识准备（已完成）
✅ 您已经有了《港险投保30问》作为理论基础

### 阶段二：产品收集（建议方式）

**方式A：官方渠道（推荐）**
1. 联系保险公司客服/顾问，索取产品说明书
2. 访问官网产品页面，下载PDF资料
3. 预约线下咨询，获取完整产品册

**方式B：半自动收集**
1. 手动访问产品页面
2. 使用浏览器开发者工具分析结构
3. 针对性地编写爬虫脚本（需要技术能力）

**方式C：逐步积累**
1. 每咨询一个产品就填写一份模板
2. 建立自己的产品信息库
3. 定期更新（产品会变化）

### 阶段三：系统对比
1. 加载所有收集的产品信息
2. 按产品类型分类对比
3. 导出Excel进行深度分析

### 阶段四：家庭配置规划
1. 输入家庭财务情况
2. 获取配置优先级建议
3. 选择最匹配的产品组合

## ⚠️ 重要提醒

### 合法合规使用
1. **仅用于个人学习研究**，不用于商业目的
2. **尊重网站服务条款**，不进行大规模自动化爬取
3. **版权尊重**：产品说明书受版权保护
4. **数据时效性**：保险产品会更新，注意标注收集日期

### 信息准确性
1. 本工具仅是组织框架，**数据准确性依赖您的输入**
2. 重要决策前务必核对官方最新资料
3. 分红、收益等非保证数字仅供参考

### 决策建议
1. 工具帮助您系统化思考，**不能替代专业顾问**
2. 港险涉及跨境、税务等复杂问题，建议咨询持牌专业人士
3. 投保前必须赴港与持牌经纪/代理人面谈

## 📞 后续行动

当您完成研究后：

1. **预约咨询**
   - 选定目标公司和产品
   - 联系持牌经纪人/代理人
   - 核查对方HKIA持牌编号

2. **赴港投保**
   - 准备身份证明、资金来源证明
   - 预留完整1天时间
   - 境内签单才合规合法

3. **后续管理**
   - 建立保单管理档案
   - 每年审视保单价值
   - 关注分红实现率公布
