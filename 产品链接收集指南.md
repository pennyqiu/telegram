## 📋 使用流程

### 第1步：找到PDF链接（5-10分钟/公司）

#### AXA安盛（最简单）

1. 访问：https://www.axa.com.hk/zh
2. 选择产品类别（如"储蓄计划"）
3. 点击具体产品（如"盛利II"）
4. 在产品页面找到"产品说明书"或"产品小册子"
5. 右键点击 → "复制链接地址"

**示例链接格式：**
```
https://www.axa.com.hk/zh/wealth-ahead-ii-savings-insurance-supreme/documents/product-brochure.pdf
```

#### Prudential保诚

1. 访问：https://www.prudential.com.hk/sc/
2. 浏览产品 → 选择类别
3. 进入产品页面
4. 下载"产品概要"或"产品说明书"
5. 复制下载链接

#### 平安保险

1. 访问：https://www.pingan.com/official/lifeinsurance
2. 选择产品
3. 查看产品详情
4. 下载"产品条款"或"产品说明书"

**技巧：**
- 有些页面PDF是弹窗打开的，可以在弹窗中复制链接
- 使用浏览器开发者工具（F12）→ Network标签页，可以看到实际的PDF请求URL

#### 中国人寿

1. 访问：http://www.chinalife.com.cn/chinalife/cpzx17/
2. 选择产品分类
3. 点击具体产品
4. 下载产品条款或说明书

## 🔍 常见问题

### Q1: 找不到PDF下载链接怎么办？

**A:** 有些产品页面PDF是动态加载的：
1. 点击"下载"按钮后，观察浏览器下载
2. 在下载管理器中复制链接
3. 或者先下载到电脑，再手动整理

### Q2: 下载的PDF是乱码？

**A:** 可能是编码问题：
1. 尝试使用其他PDF阅读器
2. 确认下载完整（检查文件大小）
3. 有些PDF可能需要密码（联系公司获取）

### Q3: 可以直接爬取整个网站吗？

**A:** 技术上可行，但**强烈不推荐**：
1. 违反网站服务条款
2. 可能被封IP
3. 法律风险
4. 半自动化方案已经足够高效

### Q4: 下载后如何与模板关联？

**A:** 下载PDF后：
1. 阅读PDF获取详细信息
2. 填写 `insurance_template.json`
3. 在模板中记录PDF文件路径
4. 使用对比工具分析

### Q5: 多久更新一次产品信息？

**A:** 建议：
- 每季度：检查关注的产品是否有更新
- 每年：重新下载主要产品说明书
- 投保前：务必下载最新版本
