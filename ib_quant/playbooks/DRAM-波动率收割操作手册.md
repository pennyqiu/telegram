## 三、为什么"价值底测算"在这里失效

QCOM 飞轮的根基是能算出价值底（14x周期PE → $150），让卖Put盈亏平衡锚定其上。**DRAM 用不了这套**：

1. 一篮子**强周期存储股**，处在超级周期**顶部**（刚涨40%）。
2. 存储是半导体里**周期性最猛**的环节，巅峰盈利→反转可跌 50–70%。
3. ETF 无 PE/DCF 公允底；上市仅2个月，无历史数据算 IV Rank。

**结论**：被行权时你无法判断接在"便宜"还是"半山腰"。因此 DRAM 只能当波动率交易，不能当价值建仓。

## 四、优缺点全景

### ✅ 优点
1. **分散单一公司暴雷风险**：无 QCOM "苹果抽走基带"式单点利空，一篮子存储龙头。
2. **IV 极端高 → 权利金极肥**：存储板块IV处历史极值（美光 IVR 100、IV 107%），ETF期权IV 100%+。
3. **底层流动性好**：日均成交 ~4700万股，AUM $233亿。
4. **单价低、资金效率高**：~$70/股，单张Put担保 ~$7000（QCOM ~$16000）。
5. **长期叙事强**：HBM/DRAM 是 AI 算力真实瓶颈。

### ⚠️ 缺点 / 风险（更关键）
1. **🔴 期权合约层面流动性差**：正股活跃但期权链薄，成交量/未平仓低、**买卖价差宽**。宽价差直接吃掉飞轮利润——对机械滚动策略是致命伤。**（首要否决项）**
2. **🔴 无价值底可锚定**：算不出公允底。
3. **🔴 极端周期性 + 抛物线动能**：1月+40%是euphoria，卖Put可能接在见顶反转点。肥权利金=崩盘风险定价，非免费午餐。
4. **新基金无历史**：未经历回撤，行为不可预测。
5. **Swap+外股结构**：对手方风险、跟踪误差、主动管理不确定性。
6. **集中度高**：前2名占33%，非真分散。

## 六、执行方案（若仍要做）

> 前提：把它当**波动率交易**，不是价值建仓。

### 开仓前硬性检查（任一不满足则放弃）
- [ ] **期权流动性**：目标strike的成交量、未平仓是否足够；**买卖价差是否 < 权利金的5%**（一票否决）
- [ ] 实时IV/IV性质（以券商数据为准，不用陈旧/过期合约报价）
- [ ] 确认这是小仓位投机，不是核心配置

### Sell Put 参数
| 参数 | 设定 |
|------|------|
| 行权价 | **深虚值（离现价 -25% 以上）**，靠极高IV照样收可观权利金 + 留足缓冲 |
| 到期 | 30–45天 |
| 仓位 | **极小，1张试水**，绝不重仓 |
| 挂单 | 限价中间价，价差宽则放弃，绝不市价吃单 |

### 风控（与飞轮不同）
1. **机械止损**：正股放量跌破关键均线/支撑即平仓认亏，**不接货长拿**。
2. **不向下展期接飞刀**：周期股下行是腰斩级，不要用"摊低成本"心态死扛。
3. **Vega 总量**：DRAM极端IV会放大账户负Vega，全市场恐慌时多仓齐跌——务必给总Vega预算留空间。
4. **被行权后**：尽快了结或卖很远的Covered Call，不当长期底仓持有。

## 七、关键提醒

- **肥权利金 = 崩盘风险定价**。DRAM 给的超高权利金，对价是"接在记忆体周期顶部、且期权不好平仓"的双重风险。
- **这不是飞轮，是波动率投机**。QCOM 那套"愿意在价值底长期持有"的心态，在 DRAM 上会让你在周期反转时严重套牢。
- **首要前提永远是期权流动性**。ETF 再热，期权价差宽就一票否决。

## 八、操作清单（Checklist）

**开仓前**
- [ ] 券商实时核对期权成交量/未平仓/买卖价差（价差>权利金5%→放弃）
- [ ] 确认极小仓位、投机定位
- [ ] 确认账户总Vega仍有预算

**卖 Put**
- [ ] 深虚值（-25%+）、30–45天、1张试水
- [ ] 限价中间价，价差宽即放弃

**持仓管理**
- [ ] 权利金缩水50–60%→止盈平仓
- [ ] 正股破关键均线→机械止损，不接飞刀
- [ ] 若被行权→尽快了结或卖远端CC，不长期持有
