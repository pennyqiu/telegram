这是什么？

一套接入 IB（盈透证券） 的量化交易系统：自动按策略盯盘、产生买卖信号、 经过多重安全护栏后下单（正股/期权），并把每一步通过 Telegram / 邮箱通知你。 本页面是它的实时看板。

怎么用起来
  1. 按 guides/01-account-setup.md 开通 IB 权限与市场数据。
  2. 按 guides/02-api-deployment.md 部署 IB Gateway + 本程序（docker compose up -d）。
  3. 保持 TRADING_MODE=paper + DRY_RUN=true，访问 http://主机:8800 观察信号与通知。
  4. 确认无误后逐步放开：模拟盘真实下单 → 实盘（收紧护栏阈值）。

⚠️ 本系统会真实下单，仅供技术研究，不构成投资建议。
