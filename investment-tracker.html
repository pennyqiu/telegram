📋 自动生成简报（VPS 定时任务）

由 VPS 每日/每周/每月自动运行 tracker.py 生成，包含 Google News 新闻抓取（浏览器版因 CORS 无法直接抓取新闻）。

📅 每日简报 每天 16:00 自动更新 📊 每周简报 每周一 16:00 自动更新 📈 月度简报 每月1日 16:00 自动更新 📁 历史快照 所有历史简报
💡 首次访问需要输入账号密码（admin / admin123），由 Nginx Basic Auth 保护
🤖 本地自动化脚本（tracker.py）

在本地运行 Python 脚本，自动生成更完整的简报（含 Google News 新闻抓取），建议配合 cron 定时执行：

# 安装依赖
pip install yfinance requests feedparser

# 每日简报（持仓涨跌 + 新闻 + 财报预警）
python investment_tracker/tracker.py daily

# 每周简报
python investment_tracker/tracker.py weekly

# 再平衡检查
python investment_tracker/tracker.py rebalance

# macOS 定时（每天 8:00 自动生成）：crontab -e 添加：
0 8 * * * cd ~/path/to/project && python investment_tracker/tracker.py daily

💡 先在 investment_tracker/tracker.py 的 PORTFOLIO 里填写你的 shares（股数），才能计算再平衡和总市值
